# Добавление фотографий

Поддерживаемые форматы: `.jpg`, `.jpeg`, `.png`, `.webp`.

Рекомендуемый размер: 1600-2400 px по длинной стороне. Для быстрой загрузки лучше заранее сжимать большие оригиналы.

Каждая фотография описывается объектом в `assets/gallery-data.js`:

```js
{
  src: "assets/photos/file-name.jpg",
  title: "Короткое название",
  alt: "Что изображено на фотографии",
  location: "Место съемки",
  date: "2026",
  tags: ["тег"]
}
```

Теги автоматически превращаются в фильтры над галереей.
